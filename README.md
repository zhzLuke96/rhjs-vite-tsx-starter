# Rhjs TypeScript(jsx) with Vite

This is a [Rh.js](https://github.com/zhzLuke96/rh.js) + [TypeScript](https://www.typescriptlang.org/) boilerplate built with [Vite](https://vitejs.dev).

# Getting started

## init
```
yarn
```

## build
```
yarn build
```

## development
```
yarn dev
```
